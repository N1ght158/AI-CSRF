# AI-CSRF

这是一个用于自动化修复 CSRF 风险的项目，关于前后端仓库的 AI CSRF 自动修复插件：自动分析、修复、测试、提关联 PR，并仅在检查通过时自动合并。

目标：
1. 接收前后端仓库地址。
2. 分析 CSRF 防护缺口。
3. 生成修复改动与测试。
4. 自动提交 PR，并在满足策略时合并。


项目文档见 `docs/` 目录。

目前:
- 目标 1：已完成。支持接收前后端仓库地址、地址校验、run-id 生成。
- 目标 2：未完成。CSRF 防护缺口识别能力还未接入。
- 目标 3：未完成。自动生成修复改动与测试还未接入。
- 目标 4：未完成。自动提交 PR 和自动合并还未接入（目前只有流程参数框架）。
