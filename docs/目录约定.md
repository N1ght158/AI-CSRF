# 目录约定

- `docs/`：方案说明、设计记录、决策文档。
- `src/`：核心实现代码。
    src/csrf_autopilot.py：只保留入口。
    src/ai_csrf/cli.py：命令行参数解析。
    src/ai_csrf/app.py：主流程编排。
    src/ai_csrf/config.py：运行配置。
    src/ai_csrf/git_client.py：Git 命令封装。
    src/ai_csrf/repository.py：仓库地址、目录、分支准备。
    src/ai_csrf/checks.py：环境检查。
    src/ai_csrf/csrf_analyzer.py：CSRF 风险扫描。
    src/ai_csrf/reports.py：报告生成。
    src/ai_csrf/rules.py：扫描规则定义。
- `scripts/`：本地辅助脚本。
- `reports/`：执行报告与结果产物。
- `repos/`：运行时拉取的目标仓库目录。
